Synchronization is a theme based on Underscores - designed to be accessible and fully responsive with a mobile-first approach. The header is fully customizable with colors, a background image and site logo. 
A starter theme called synchronization.

== Features ==

=== Customizer Options ===
Extended options available from the Customizer:

1. Optional Logo to the left of the site title/description in header
2. Header background color
3. Header text color
4. Optional Header image
5. Sidebar position: right.

=== Featured Images ===
Posts and pages can have Featured Images.

=== Widgetized Areas ===
1. Sidebar
2. Foooter
